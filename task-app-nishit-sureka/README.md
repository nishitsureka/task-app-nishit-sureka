## Setup
1. composer install,
2. php artisan db:seed,
3. php artisan serve

## NPm
1. npm install vue@3,
2. npm install @vitejs/plugin-vue --save-dev,
3. npm run dev

## Run Tests
1. php artisan test

## API Endpoints
1. GET /api/tasks,
2. POST /api/tasks,
3. PUT /api/tasks/{id},
4. DELETE /api/tasks/{id}
